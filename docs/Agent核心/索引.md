# Agent 核心

本目录包含 ForgeAgent Agent 的核心模块文档。

## 四大模块

| 模块 | 核心功能 |
|------|----------|
| **规划** | 任务分解、计划生成、重规划 |
| **执行** | ReAct 循环、工具调用、结果汇总 |
| **记忆** | 会话记忆、黑板反思、上下文 |
| **上下文** | Token 预算、截断策略、摘要压缩 |
