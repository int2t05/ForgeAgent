## Learner 反思

```python
async def learn_node(state: AgentState) -> dict:
    # 1. 生成反思
    reflection = await llm.generate_reflection(
        tool_trace=state["actor_tool_trace"],
        outcome=state["outcome"]
    )

    # 2. 追加到黑板
    notes = list(state["blackboard_notes"])
    notes.append(reflection["text"])

    # 3. 决定是否重规划
    wants_replan = reflection.get("request_replan")

    return {"blackboard_notes": notes, "replan_requested": wants_replan}
```

## RAG 知识库

基于 LangChain LCEL 构建的 RAG 系统，支持文档分块、向量化、混合检索和 Rerank 重排序。

### 核心组件

| 组件 | 说明 |
|------|------|
| `RagKnowledgeBase` | RAG 知识库管理器 |
| `HybridSearch` | 混合检索器（向量 + BM25） |
| `BM25Retriever` | BM25 关键字检索器 |
| `get_rag_chain()` | LCEL RAG Chain 构建器 |

### 使用示例

```python
from app.modules.memory.rag import RagKnowledgeBase, RagConfig, get_rag_chain

# 创建知识库
config = RagConfig(
    persist_directory="./db_rag",
    chunk_size=500,
    chunk_overlap=50,
)
rag = RagKnowledgeBase(config)

# 摄入文档
await rag.aingest_text(
    "这是要摄入的文档内容...",
    metadata={"source": "README", "author": "system"}
)

# 检索
results = await rag.asearch("查询语句", top_k=5)
for r in results:
    print(f"内容: {r.content}, 分数: {r.score}, 来源: {r.source}")

# 构建 RAG Chain
chain = get_rag_chain(rag, llm)
result = await chain.ainvoke({"question": "什么是 ForgeAgent？"})
```

### 混合检索流程

```
1. 向量检索 → Top-K 向量相似度结果
2. BM25 检索 → Top-K 关键字匹配结果
3. RRF 融合 → 倒数排名融合
4. Rerank → 交叉编码器重排序
5. 返回最终结果
```

### 配置

| 配置 | 默认值 | 说明 |
|------|--------|------|
| `rag_enabled` | True | 启用 RAG |
| `rag_persist_directory` | None | 向量存储持久化目录 |
| `rag_chunk_size` | 500 | 文档分块大小 |
| `rag_chunk_overlap` | 50 | 分块重叠大小 |
| `rag_embedding_model` | None | 嵌入模型 |
| `rag_vector_weight` | 0.5 | 向量检索权重 |
| `rag_bm25_weight` | 0.5 | BM25 权重 |
| `rag_default_top_k` | 5 | 默认返回数量 |

## 配置

| 配置 | 默认值 | 说明 |
|------|--------|------|
| `session_memory_max_messages` | 32 | 消息加载上限 |
| `session_blackboard_max_notes` | 64 | 黑板最大条数 |
| `session_conversation_summary_enabled` | True | 启用摘要 |
| `session_summarize_when_over` | 20 | 触发摘要阈值 |
