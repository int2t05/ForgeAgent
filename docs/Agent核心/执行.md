## 工具执行

```python
async def run_single_tool_with_retry(task_id, tool_name, args, max_tries):
    for attempt in range(max_tries):
        result = await tool_registry.execute(tool_name, args)
        if result["ok"]:
            return result
        # 指数退避重试
```
