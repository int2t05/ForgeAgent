## 架构分层

```
┌─────────────────────────────────────────┐
│              Frontend (React)             │
├─────────────────────────────────────────┤
│              Backend (FastAPI)            │
│  ┌─────────────────────────────────┐   │
│  │     API Layer → Service Layer    │   │
│  ├─────────────────────────────────┤   │
│  │   Agent Modules (LangGraph)      │   │
│  │   规划 / 执行 / 记忆 / RAG       │   │
│  ├─────────────────────────────────┤   │
│  │        Repository Layer          │   │
│  └─────────────────────────────────┘   │
├─────────────────────────────────────────┤
│              SQLite Database              │
└─────────────────────────────────────────┘
```

## 核心模块

| 模块 | 职责 |
|------|------|
| 规划 | 任务分解、计划生成、RAG 检索 |
| 执行 | ReAct 循环、工具调用 |
| 记忆 | 会话记忆、黑板反思、RAG 知识库 |
| 接口 | REST API、SSE 事件流 |
