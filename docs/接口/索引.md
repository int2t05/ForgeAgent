# 接口文档

本目录包含 ForgeAgent REST API 和 SSE 接口文档。

## 基础信息

| 项目 | 值 |
|------|---|
| Base URL | `http://localhost:8000` |
| API 文档 | `http://localhost:8000/docs` |
| OpenAPI JSON | `http://localhost:8000/openapi.json` |

## API 分类

| 分类 | 端点数 | 说明 |
|------|--------|------|
| Session | 4 | 会话管理 |
| Task | 6 | 任务 CRUD + SSE |
| Tool | 1 | 工具列表 |
| Settings | 4 | 用户设置 |

## SSE 事件流

```
GET /api/v1/tasks/{id}/events/stream
```

返回 `text/event-stream`，支持事件：

| 事件 | 说明 |
|------|------|
| `plan_created` | 计划已生成 |
| `step_start` | 步骤开始 |
| `tool_call` | 工具调用 |
| `tool_result` | 工具结果 |
| `step_end` | 步骤结束 |
| `message` | 助手消息 |
| `task_complete` | 任务完成 |
