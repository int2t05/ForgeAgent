## 2. 任务执行 (Task Execution)

### 2.1 创建并启动任务
```python
POST /api/v1/tasks
{
    "session_id": str,
    "user_message": str,
    "reuse_user_message_id": int | null   # 可选：复用已有用户消息重跑
}

# 语义摘要（以 task_service / OpenAPI 为准）
# 1) 校验会话；2) 写入或复用用户消息；3) 创建 tasks 行；4) asyncio.create_task 执行 LangGraph
# 5) 返回 { "task_id", "events_stream_path" }
```

### 2.2 LangGraph 执行流程（Plan-Act-Learn）

```text
START → planner → actor → learner → 条件边
  - 若 outcome == failed → END
  - 若 replan_requested 且未超 max_replan_attempts → planner（在 planner 入口 bump plan_version + task_events.replan）
  - 否则 END
```

运行时使用 **`graph.astream(input, config, stream_mode="updates")`**，每次节点完成将增量映射为 **`task_events`**（`module=workflow`, `kind=node_update`）。**`configurable.thread_id`** 与 **`task_id`** 对齐，检查点落在独立 SQLite（`LANGGRAPH_CHECKPOINT_SQLITE_PATH`）或可选 Postgres。

### 2.3 任务节点概要

| 节点 | 职责摘要 |
|------|----------|
| `planner` | `SessionLLMContextManager` 加载会话消息；拼接黑板要点；`plan_steps_with_llm`；写 `plan_created` |
| `actor` | 逐步 `tool_registry.execute`；`tool_call`/`tool_result`/`step_end`；流式总结 `llm_stream_delta` |
| `learner` | 反思与黑板更新；设置 `replan_requested` 或失败短路 |

### 2.4 查询任务
```python
GET /api/v1/tasks?status=running&limit=20&offset=0
GET /api/v1/tasks/{task_id}

# 伪代码
def list_tasks(status: str | None, limit: int, offset: int):
    query = db.query(Task)
    if status:
        query = query.filter(Task.status == status)
    return query.order_by(Task.created_at.desc()).limit(limit).offset(offset).all()

def get_task(task_id: str):
    return db.get(Task, task_id)
```

## 3. SSE 事件流 (Event Streaming)

### 3.1 事件流端点
```python
GET /api/v1/tasks/{task_id}/events/stream?after_seq=0

# 伪代码
async def stream_task_events(task_id: str, after_seq: int = 0):
    last_seq = after_seq
    empty_rounds = 0

    while True:
        events = await db.query(TaskEvent)
            .filter(TaskEvent.task_id == task_id)
            .filter(TaskEvent.seq > last_seq)
            .order_by(TaskEvent.seq)
            .all()

        if events:
            empty_rounds = 0
            for event in events:
                yield f"event: {event.kind}\ndata: {event.payload_json}\n\n"
                last_seq = event.seq
        else:
            empty_rounds += 1

        # 终端状态4轮轮询后关闭
        task = await db.get(Task, task_id)
        if task.status in ("success", "failed", "cancelled") and empty_rounds >= 4:
            break

        await asyncio.sleep(0.15)  # 轮询间隔
```

### 3.2 写入任务事件
```python
# 伪代码
async def write_event(task_id: str, kind: str, payload: dict):
    seq = await db.execute(
        select(func.count()).where(TaskEvent.task_id == task_id)
    ) + 1

    event = TaskEvent(
        task_id=task_id,
        seq=seq,
        ts=now(),
        module=payload.get("module", "execution"),
        kind=kind,
        payload_json=json.dumps(payload)
    )
    await db.add(event)
    await db.commit()
    return event
```

## 4. 工具注册 (Tool Registry)

### 4.1 获取所有工具
```python
GET /api/v1/tools

# 伪代码
def get_all_tools() -> List[dict]:
    tools = []

    # 内置工具
    tools.extend([
        {"name": "duckduckgo_search", "description": "...", "module": "builtin"},
        {"name": "list_tools", "description": "...", "module": "builtin"},
    ])

    # MCP 工具 (从设置加载)
    mcp_config = get_settings("mcp")
    for server_name, tools_list in mcp_config.get("tools", {}).items():
        for tool in tools_list:
            tools.append({**tool, "module": "mcp", "server": server_name})

    # Skill 工具 (从配置的路径加载)
    skills_paths = get_settings("skills_paths", [])
    for path in skills_paths:
        tools.extend(load_skill_tools(path))

    return tools
```

### 4.2 工具执行
```python
# 伪代码
async def execute_tool(name: str, args: dict) -> dict:
    tool = registry.get(name)
    if not tool:
        return {"error": f"Tool '{name}' not found"}

    if tool["module"] == "builtin":
        return await builtin_tools[name](args)
    elif tool["module"] == "mcp":
        return await mcp_tools[name](args)
    elif tool["module"] == "skill":
        return await skill_tools[name](args)
```

### 5.2 更新设置
```python
PUT /api/v1/settings
{
    "mcp": {...},
    "skills_paths": [...]
}

# 伪代码
def update_settings(settings: dict):
    for key, value in settings.items():
        # 安全检查: 禁止包含敏感关键词
        if any(s in key.lower() for s in ["api_key", "secret", "token", "password"]):
            raise HTTPException(400, f"Key '{key}' contains disallowed fragment")

        await db.merge(SettingsKV(
            key=key,
            value_json=json.dumps(value),
            updated_at=now()
        ))
    await db.commit()

    # 刷新工具注册表
    refresh_tool_registry()
```
