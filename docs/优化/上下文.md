# 上下文管理优化

## 现有机制

| 模块 | 文件 | 功能 |
|------|------|------|
| 消息加载 | `memory/context.py` | 从 DB 加载最近 32 条 |
| Token 估算 | `memory/context.py` | tiktoken / 模型计数 |
| 截断策略 | `memory/context.py` | 保留系统块，自新向旧填充 |
| 黑板记忆 | `shared/blackboard.py` | 跨任务持久化 |
| RAG 知识库 | `memory/rag.py` | 向量检索长期记忆 |

## 优化方案

### Tier 1: 精确 Token 计数

```python
from app.modules.memory.context import estimate_messages_tokens

# 自动选择最佳计数方式
count = estimate_messages_tokens(chat_model, messages)
```

### Tier 2: 对话摘要压缩

```python
from app.modules.memory.context import maybe_compress_chat_history

# 当消息条数超过阈值时自动压缩
compressed = await maybe_compress_chat_history(messages, settings)
```

### Tier 3: RAG 长期记忆

```python
from app.modules.memory.rag import RagKnowledgeBase

# 摄入文档作为长期记忆
await rag.aingest_text(
    "项目文档内容...",
    metadata={"source": "README", "type": "knowledge"}
)

# 检索相关记忆
results = await rag.asearch("用户偏好")
```

### Tier 4: 结构化黑板

```python
from app.shared.blackboard import cap_blackboard_notes, encode_blackboard_json

# 截断并编码黑板
notes = cap_blackboard_notes(notes, max_notes=64)
encoded = encode_blackboard_json(notes, max_notes=64)
```

## 实施优先级

| 优先级 | 优化项 | 效果 |
|--------|--------|------|
| P0 | 精确 Token 计数 | 截断精度 95%+ |
| P0 | 智能截断策略 | 保留更多上下文 |
| P1 | 对话摘要压缩 | 长对话不丢历史 |
| P1 | RAG 知识库检索 | 长期记忆可检索 |
| P2 | 跨会话长期记忆 | 跨会话偏好 |
