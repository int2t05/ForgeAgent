# 优化文档

本目录包含 ForgeAgent 性能与效果优化方案文档。

## 优化分类

| 分类 | 目标 |
|------|------|
| **性能** | 降低延迟、减少资源占用 |
| **提示词** | 提升输出质量、可维护性 |
| **上下文** | 优化上下文利用效率（Token、RAG） |

## 文档

| 文档 | 说明 |
|------|------|
| [性能.md](性能.md) | DB 优化、前端优化、执行优化 |
| [提示词.md](提示词.md) | 模板化、版本管理、动态注入 |
| [上下文.md](上下文.md) | Token 计数、摘要压缩、记忆管理、RAG |

## 优化优先级

| 优先级 | 问题 | 影响 |
|--------|------|------|
| P0 | DB 查询风暴 | SSE 期间每秒多次查询 |
| P0 | 频繁创建 DB 连接 | 每个任务 13+ 次连接 |
| P1 | 前端重渲染 | SSE 事件触发全组件更新 |
| P2 | ReAct 串行执行 | 工具间无并行 |
