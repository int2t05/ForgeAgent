## 面试重点

### 必须掌握（难度 ★★★）

1. **Plan-Act-Learn 三角循环**
   - 三个节点的职责和数据流
   - 条件边路由逻辑
   - 重规划触发条件

2. **LangGraph Checkpoint**
   - 为什么独立 SQLite 存储
   - 如何实现服务重启后恢复
   - thread_id 与 task_id 的关系

3. **ReAct 循环**
   - 思考→行动→观察循环流程
   - LLM 输出格式解析
   - 工具调用带重试机制

### 深入理解（难度 ★★★★）

4. **Token 管理**
   - tiktoken 计数原理
   - 截断算法细节
   - 对话摘要压缩流程

5. **RAG 混合检索**
   - BM25 公式推导
   - RRF 融合算法
   - LCEL RAG Chain 构建

6. **熔断器模式**
   - 三态机转换
   - 失败计数与超时机制
   - 配置阈值建议

## 面试技巧

### 项目介绍模板

```
ForgeAgent 是一个基于 Python + FastAPI + LangGraph 构建的企业级 AI Agent 应用。

核心特性：
1. Plan-Act-Learn 三角循环：Planner 负责规划、Actor 负责执行、Learner 负责反思
2. RAG 知识库：向量 + BM25 混合检索 + Rerank 重排序
3. 记忆系统：会话黑板跨任务共享、Token 预算控制、对话摘要压缩
4. 工程稳定性：SSE 流式输出、熔断器保护、检查点持久化

技术栈：Python 3.11+ / FastAPI / LangGraph / LangChain / React 18 + TypeScript / SQLite
```

```
这个问题可以分为几个层面来考虑：

1. 需求分析

2. 技术选型

3. 实现细节
```
